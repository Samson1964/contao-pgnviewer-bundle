# PGN-Betrachter

Der PGN-Betrachter stellt ein Inhaltselement PGNViewer zur Verfügung. Als Engine zur Anzeige der Partien wird der PGN-Viewer von Chesstempo verwendet, der für die Verwendung in Contao leicht modifiziert wurde: bei den Dateinamen und den Pfaden zu den Dateien.
